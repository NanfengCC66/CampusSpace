# CampusSpace 冲突检测规则

**版本**：v1.1  
**更新日期**：2026-07-17

---

## 冲突类型定义

| 冲突类型 | 英文 | 说明 | 优先级 |
|---------|------|------|--------|
| 维修停用冲突 | maintenance | 预约时间与维修停用时间重叠 | 最高 |
| 固定课表冲突 | schedule | 预约时间与固定课表时间重叠 | 高 |
| 临时预约冲突 | booking | 预约时间与已通过预约时间重叠 | 中 |
| 时间间隔冲突 | interval | 预约时间间隔不足（前后预约时间太近） | 低（强制） |

**优先级规则**：维修停用 > 固定课表 > 临时预约

---

## 冲突检测流程

```
输入：场地ID、开始时间、结束时间
  ↓
检测维修停用冲突
  ↓ 有冲突 → 返回冲突信息
  ↓ 无冲突
检测固定课表冲突
  ↓ 有冲突 → 返回冲突信息
  ↓ 无冲突
检测临时预约冲突
  ↓ 有冲突 → 返回冲突信息
  ↓ 无冲突
检测时间间隔冲突（强制）
  ↓ 有冲突 → 返回冲突信息
  ↓ 无冲突
返回：无冲突，可预约
```

## 1. 维修停用冲突检测

### 检测规则

**规则描述**：场地在维修停用期间不可预约

**检测条件**：
```
存在维修记录满足：
  venue_id = 目标场地ID
  AND status = 'ongoing'
  AND (
    (start_time <= 预约开始时间 < end_time)
    OR (start_time < 预约结束时间 <= end_time)
    OR (预约开始时间 <= start_time AND 预约结束时间 >= end_time)
  )
```

**简化判断**：
```
两个时间段重叠 ⟺ max(开始时间1, 开始时间2) < min(结束时间1, 结束时间2)
```

### 代码实现

```python
def check_maintenance_conflict(venue, start_time, end_time):
    """
    检测维修停用冲突
    
    Args:
        venue: 场地对象
        start_time: 预约开始时间
        end_time: 预约结束时间
    
    Returns:
        None 或 冲突信息字典
    """
    # 查询进行中的维修记录
    maintenances = Maintenance.objects.filter(
        venue=venue,
        status='ongoing'
    )
    
    for maintenance in maintenances:
        # 判断时间是否重叠
        if has_time_overlap(
            start_time, end_time,
            maintenance.start_time, maintenance.end_time
        ):
            return {
                'type': 'maintenance',
                'message': f'场地在{maintenance.start_time}至{maintenance.end_time}维修停用',
                'detail': {
                    'reason': maintenance.reason,
                    'start_time': maintenance.start_time,
                    'end_time': maintenance.end_time,
                }
            }
    
    return None


def has_time_overlap(start1, end1, start2, end2):
    """
    判断两个时间段是否重叠
    """
    return max(start1, start2) < min(end1, end2)
```

## 2. 固定课表冲突检测

### 检测规则

**规则描述**：场地在固定课表时间段不可临时预约

**检测条件**：
```
存在课表记录满足：
  venue_id = 目标场地ID
  AND semester = 当前学期
  AND 预约时间在课表周次范围内
  AND 预约日期的周几 = 课表的day_of_week
  AND 预约时间段与课表节次重叠
```

### 时间转换

**需要将预约时间转换为周次和节次**：

```python
def get_week_and_day(date, semester_start_date):
    """
    获取日期对应的周次和星期几
    
    Args:
        date: 日期
        semester_start_date: 学期开始日期
    
    Returns:
        (周次, 星期几)
    """
    # 计算天数差
    days = (date.date() - semester_start_date).days
    
    # 计算周次（从1开始）
    week = days // 7 + 1
    
    # 计算星期几（1-7，周一到周日）
    day_of_week = date.isoweekday()
    
    return week, day_of_week


def get_period_from_time(time):
    """
    根据时间获取节次
    
    假设节次时间表：
    第1-2节: 08:00-09:40
    第3-4节: 10:00-11:40
    第5-6节: 14:00-15:40
    第7-8节: 16:00-17:40
    第9-10节: 19:00-20:40
    """
    period_time_map = {
        (8, 0, 9, 40): '1-2',
        (10, 0, 11, 40): '3-4',
        (14, 0, 15, 40): '5-6',
        (16, 0, 17, 40): '7-8',
        (19, 0, 20, 40): '9-10',
    }
    
    hour = time.hour
    minute = time.minute
    
    for (start_h, start_m, end_h, end_m), period in period_time_map.items():
        if start_h <= hour < end_h or (hour == end_h and minute <= end_m):
            return period
    
    return None
```

### 代码实现

```python
def check_schedule_conflict(venue, start_time, end_time, semester):
    """
    检测固定课表冲突
    
    Args:
        venue: 场地对象
        start_time: 预约开始时间
        end_time: 预约结束时间
        semester: 学期
    
    Returns:
        None 或 冲突信息字典
    """
    # 获取学期开始日期
    semester_start = get_semester_start_date(semester)
    
    # 获取预约时间的周次和星期几
    week, day_of_week = get_week_and_day(start_time, semester_start)
    
    # 查询课表
    schedules = Schedule.objects.filter(
        venue=venue,
        semester=semester,
        day_of_week=day_of_week,
        start_week__lte=week,
        end_week__gte=week
    )
    
    # 获取预约时间段对应的节次
    booking_period = get_period_from_time(start_time)
    
    for schedule in schedules:
        # 判断节次是否重叠
        if has_period_overlap(booking_period, schedule.period):
            return {
                'type': 'schedule',
                'message': f'场地在此时段有课程：{schedule.course_name}',
                'detail': {
                    'course_name': schedule.course_name,
                    'teacher': schedule.teacher,
                    'period': schedule.period,
                    'week': f'第{schedule.start_week}-{schedule.end_week}周',
                }
            }
    
    return None


def has_period_overlap(period1, period2):
    """
    判断两个节次是否重叠
    
    Args:
        period1: 如 '1-2'
        period2: 如 '3-4'
    
    Returns:
        True/False
    """
    # 解析节次
    start1, end1 = map(int, period1.split('-'))
    start2, end2 = map(int, period2.split('-'))
    
    # 判断是否重叠
    return max(start1, start2) <= min(end1, end2)
```

## 3. 临时预约冲突检测

### 检测规则

**规则描述**：场地在同一时间段不可被多个预约占用

**检测条件**：
```
存在预约记录满足：
  venue_id = 目标场地ID
  AND status = 'approved'
  AND 预约时间段重叠
  AND 预约ID != 当前预约ID（排除自己，用于修改场景）
```

### 代码实现

```python
def check_booking_conflict(venue, start_time, end_time, exclude_booking_id=None):
    """
    检测临时预约冲突
    
    Args:
        venue: 场地对象
        start_time: 预约开始时间
        end_time: 预约结束时间
        exclude_booking_id: 排除的预约ID（用于修改场景）
    
    Returns:
        None 或 冲突信息字典
    """
    # 查询已通过的预约
    bookings = Booking.objects.filter(
        venue=venue,
        status='approved'
    )
    
    # 排除指定预约
    if exclude_booking_id:
        bookings = bookings.exclude(id=exclude_booking_id)
    
    for booking in bookings:
        # 判断时间是否重叠
        if has_time_overlap(
            start_time, end_time,
            booking.start_time, booking.end_time
        ):
            return {
                'type': 'booking',
                'message': f'场地在{booking.start_time}至{booking.end_time}已被预约',
                'detail': {
                    'booking_no': booking.booking_no,
                    'title': booking.title,
                    'start_time': booking.start_time,
                    'end_time': booking.end_time,
                }
            }
    
    return None
```

## 4. 时间间隔冲突检测（强制）

### 检测规则

**规则描述**：前后预约之间需要10分钟的时间间隔（用于场地准备、清理等）

**检测条件**：
```
前一个预约：结束时间 + 10分钟 > 预约开始时间
后一个预约：预约结束时间 + 10分钟 > 预约开始时间
```

**说明**：此检测为强制要求，不满足则视为冲突

### 代码实现

```python
def check_interval_conflict(venue, start_time, end_time, interval_minutes=10):
    """
    检测时间间隔冲突（强制）
    
    Args:
        venue: 场地对象
        start_time: 预约开始时间
        end_time: 预约结束时间
        interval_minutes: 间隔分钟数（默认10分钟）
    
    Returns:
        None 或 冲突信息字典
    """
    interval = timedelta(minutes=interval_minutes)
    
    # 查询前后预约
    bookings = Booking.objects.filter(
        venue=venue,
        status='approved'
    )
    
    conflicts = []
    
    for booking in bookings:
        # 检查前一个预约
        if booking.end_time <= start_time:
            if booking.end_time + interval > start_time:
                conflicts.append({
                    'type': 'interval_before',
                    'message': f'与前一个预约间隔不足{interval_minutes}分钟',
                    'detail': {
                        'booking_no': booking.booking_no,
                        'end_time': booking.end_time,
                        'required_interval': f'{interval_minutes}分钟',
                    }
                })
        
        # 检查后一个预约
        if booking.start_time >= end_time:
            if end_time + interval > booking.start_time:
                conflicts.append({
                    'type': 'interval_after',
                    'message': f'与后一个预约间隔不足{interval_minutes}分钟',
                    'detail': {
                        'booking_no': booking.booking_no,
                        'start_time': booking.start_time,
                        'required_interval': f'{interval_minutes}分钟',
                    }
                })
    
    return conflicts if conflicts else None
```

## 综合冲突检测

### 主检测函数

```python
def check_all_conflicts(venue, start_time, end_time, 
                        semester=None, 
                        exclude_booking_id=None):
    """
    综合冲突检测
    
    Args:
        venue: 场地对象
        start_time: 预约开始时间
        end_time: 预约结束时间
        semester: 学期（用于课表检测）
        exclude_booking_id: 排除的预约ID
    
    Returns:
        {
            'has_conflict': True/False,
            'conflicts': [冲突列表]
        }
    """
    conflicts = []
    
    # 1. 检测维修停用冲突
    maintenance_conflict = check_maintenance_conflict(venue, start_time, end_time)
    if maintenance_conflict:
        conflicts.append(maintenance_conflict)
    
    # 2. 检测固定课表冲突
    if semester:
        schedule_conflict = check_schedule_conflict(venue, start_time, end_time, semester)
        if schedule_conflict:
            conflicts.append(schedule_conflict)
    
    # 3. 检测临时预约冲突
    booking_conflict = check_booking_conflict(venue, start_time, end_time, exclude_booking_id)
    if booking_conflict:
        conflicts.append(booking_conflict)
    
    # 4. 检测时间间隔冲突（强制）
    interval_conflicts = check_interval_conflict(venue, start_time, end_time)
    if interval_conflicts:
        conflicts.extend(interval_conflicts)
    
    return {
        'has_conflict': len(conflicts) > 0,
        'conflicts': conflicts,
    }
```

    return {

    }

## 冲突处理策略

### 1. 提交预约时

```python
def create_booking_with_conflict_check(user, venue, start_time, end_time, **kwargs):
    """
    创建预约（带冲突检测）
    """
    # 综合冲突检测
    result = check_all_conflicts(
        venue=venue,
        start_time=start_time,
        end_time=end_time,
        semester=get_current_semester()
    )
    
    # 有冲突则抛出异常
    if result['has_conflict']:
        conflict_messages = [c['message'] for c in result['conflicts']]
        raise ConflictError('、'.join(conflict_messages))
    
    # 有警告则记录（可选择是否继续）
    if result['warnings']:
        # 可以选择记录警告或提示用户
        pass
    
    # 创建预约
    booking = Booking.objects.create(
        venue=venue,
        user=user,
        start_time=start_time,
        end_time=end_time,
        status='pending',
        **kwargs
    )
    
    return booking
```

### 2. 审批预约时

```python
def approve_booking_with_conflict_check(booking, approver, comment=''):
    """
    审批预约（带冲突检测）
    """
    # 再次检测冲突（防止并发）
    result = check_all_conflicts(
        venue=booking.venue,
        start_time=booking.start_time,
        end_time=booking.end_time,
        semester=get_current_semester(),
        exclude_booking_id=booking.id
    )
    
    # 有冲突则拒绝
    if result['has_conflict']:
        conflict_messages = [c['message'] for c in result['conflicts']]
        raise ConflictError(f'审批时检测到冲突：{"、".join(conflict_messages)}')
    
    # 审批通过
    booking.status = 'approved'
    booking.save()
    
    # 创建审批记录
    Approval.objects.create(
        booking=booking,
        approver=approver,
        action='approve',
        comment=comment
    )
    
    return booking
```

class ConflictDetectionTestCase(TestCase):
    def test_maintenance_conflict(self):
        """测试维修停用冲突"""
        # 创建维修记录
        maintenance = Maintenance.objects.create(
            venue=self.venue,
