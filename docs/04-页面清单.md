## 1. 公共页面

### P-001: 首页

**页面名称**：首页  
**URL**：`/`  
**访问权限**：所有用户  
**页面描述**：系统首页，展示系统介绍、快速入口、公告通知

**页面元素**：
- 顶部导航栏：Logo、菜单、登录/注册按钮、用户信息
- Banner区：系统介绍、核心功能亮点
- 快速入口：场地查询、我的预约、智能推荐
- 公告通知：系统公告、使用指南
- 底部：版权信息、联系方式

**交互功能**：
- 点击快速入口跳转对应功能页面
- 未登录用户点击功能提示登录

---

### P-002: 登录页面

**页面名称**：用户登录  
**URL**：`/login/`  
**访问权限**：未登录用户  
**页面描述**：用户登录系统

**页面元素**：
- 登录表单：用户名、密码、记住我
- 登录按钮
- 注册链接
- 忘记密码链接（可选）

**交互功能**：
- 表单验证
- 登录成功跳转首页或来源页
- 登录失败显示错误提示

---

### P-003: 注册页面

**页面名称**：用户注册  
**URL**：`/register/`  
**访问权限**：未登录用户  
**页面描述**：新用户注册账号

**页面元素**：
- 注册表单：用户名、密码、确认密码、邮箱、姓名、角色（学生/教师）
- 注册按钮
- 已有账号登录链接

**交互功能**：
- 表单验证（实时验证）
- 注册成功跳转登录页
- 注册失败显示错误提示

## 2. 普通用户页面

### P-004: 场地列表页面

**页面名称**：场地查询  
**URL**：`/venues/`  
**访问权限**：已登录用户  
**页面描述**：查询和浏览场地列表

**页面元素**：
- 搜索筛选区：
  - 场地名称（模糊搜索）
  - 场地类型（下拉选择）
  - 容量范围（最小-最大）
  - 设备需求（多选）
  - 楼宇/校区（下拉选择）
  - 搜索按钮、重置按钮
- 场地列表：
  - 场地卡片：名称、类型、位置、容量、设备标签、状态
  - 查看详情按钮
  - 预约按钮
- 分页控件

**交互功能**：
- 条件筛选实时更新结果
- 点击卡片查看详情
- 点击预约跳转预约页面
- 分页浏览

---

### P-005: 场地详情页面

**页面名称**：场地详情  
**URL**：`/venues/<id>/`  
**访问权限**：已登录用户  
**页面描述**：查看场地详细信息和占用情况

**页面元素**：
- 场地基本信息：
  - 名称、类型、位置（楼宇、楼层、房间号）
  - 容量、面积
  - 设备清单（图标+文字）
  - 场地图片（轮播）
  - 负责人、联系方式
  - 备注
- 日历视图标签页：
  - 周视图/月视图切换
  - 显示固定课表、已审批预约、维修停用
  - 不同类型用不同颜色标识
  - 点击占用块显示详情
- 操作按钮：
  - 预约此场地
  - 返回列表

**交互功能**：
- 切换日历视图类型
- 点击日历事件查看详情
- 点击预约跳转预约页面

---

### P-006: 智能推荐页面

**页面名称**：智能推荐  
**URL**：`/recommend/`  
**访问权限**：已登录用户  
**页面描述**：根据需求智能推荐场地

**页面元素**：
- 需求输入区：
  - 使用时间：开始时间、结束时间
  - 参与人数
  - 设备需求（多选）
  - 其他偏好：楼宇、距离等（可选）
  - 推荐按钮
- 推荐结果区：
  - 推荐场地卡片：
    - 场地名称、位置、容量
    - 匹配度评分（如：95%）
    - 推荐理由（如："容量完全匹配，设备齐全"）
    - 设备匹配情况
    - 查看详情按钮
    - 立即预约按钮
- 无结果提示：
  - 未找到完全匹配的场地
  - 推荐相近场地（如有）

**交互功能**：
- 提交需求获取推荐
- 查看推荐场地详情
- 直接预约推荐场地

---

### P-007: 提交预约页面

**页面名称**：提交预约  
**URL**：`/booking/create/?venue=<id>`  
**访问权限**：已登录用户  
**页面描述**：填写预约信息并提交

**页面元素**：
- 场地信息（只读）：
  - 场地名称、位置、容量
- 预约表单：
  - 使用时间：开始时间、结束时间（日期时间选择器）
  - 参与人数（数字输入，不超过场地容量）
  - 使用目的/活动名称（文本）
  - 联系方式（电话）
  - 备注（文本域，可选）
  - 提交按钮
  - 取消按钮

**交互功能**：
- 时间选择器选择日期时间
- 表单验证（实时验证）
- 提交前冲突检测
- 提交成功跳转预约详情页

---

```
首页 (P-001)
  ├─ 登录 (P-002) → 首页
  ├─ 注册 (P-003) → 登录

       └─ 智能推荐 (P-006)
            └─ 提交预约 (P-007) → 预约详情 (P-009)
  
我的预约 (P-008)
  └─ 预约详情 (P-009)

管理员首页 (P-010)
  ├─ 场地管理 (P-011)
  │    ├─ 添加场地 (P-012)
  │    └─ 编辑场地 (P-012)
  ├─ 课表管理 (P-013)
  │    ├─ 添加课表 (P-014)
  │    └─ 编辑课表 (P-014)
  ├─ 维修管理 (P-015)
  │    └─ 添加维修 (P-016)
  ├─ 预约审批 (P-017)
  ├─ 统计分析 (P-018)
  └─ 预约管理 (P-019)
```
